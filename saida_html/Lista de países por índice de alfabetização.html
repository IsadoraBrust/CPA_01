Este artigo é uma lista de países por índice de alfabetização. Os dados são provenientes do CIA World Factbook,[1] e dados nacionais oficiais. Em casos de indisponibilidade destas informações foram usados dados antigos. Para países altamente desenvolvidos onde as estatísticas não foram recolhidas, presume-se uma taxa de 99%.

Lista

[editar | editar código-fonte]
